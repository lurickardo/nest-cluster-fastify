## Description

Application in Nestjs using Node Cluster and Fastify, documented with swagger.

# Libraries

- NodeJS
- Fastify
- Swagger
- Typescript

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Swagger

`http://localhost:3000/api/nest-cluster-fastify/docs`
